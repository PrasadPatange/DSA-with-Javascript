Javascript Object

Object.assign() :

It Copies all enumerable own properties from one or more source object to a target object.

for...in :

Creates loop iteration over property names in the object.

for...of :

Creates loop iteration over iterable object values.

Object.keys() :

Returns the names of the enumerable string properties and methods of an object.

Object.values() :

Returns an array of values of the enumerable properties of an object
