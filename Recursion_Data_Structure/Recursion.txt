4) Head and Tail Recursion :-
    1) Head Recursion :
     -> Recursion function before condition
      Ex :
        function test(x){
         console.log(x); //before condition
         if(x>0){
            test(x-1);
         }
        }
        test(5);
    2) Tail Recursion :
     -> Recursion function after condition
           Ex :
        function test(x){
         if(x>0){
            test(x-1);
         }
         console.log(x); // After condition
        }
        test(5);
